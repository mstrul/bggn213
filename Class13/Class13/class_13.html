  • 1.) Read input files,
    • a.) countdata
    • b.) coldata
  • 2.) Check and fix
    • a.) remove zero count genes across all
  • 3.)DESEQ
    • a.) plot of Log2fc vs -log(p-value)
    • b.)write csv of results
  • 4.) Annotation
  • 5.) Pathway analysis
